# JetBoil MiniMo Field Cookbook

This is a structured field-cooking knowledge base for the JetBoil MiniMo, using both the MiniMo pot and JetBoil skillet.

The goal is not merely to create one cookbook, but to maintain a detailed canonical database that can later generate:

- Minimal field books
- Standard recipe books
- Comprehensive reference manuals
- Recipe cards
- Shopping lists
- Pantry modules
- Meal plans
- Searchable databases
- Printable PDFs

Created pilot package: 2026-05-18
