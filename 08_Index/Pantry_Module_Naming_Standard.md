# Pantry Module Naming Standard

## General Rules

1. Use clear functional names.
2. Avoid overly clever names.
3. Use singular `Module` for functional modules.
4. Use `Pantry` for trip-based or broad supply modules.
5. Keep names short enough to be useful in file names and links.

## Preferred Names

| Preferred | Avoid |
|---|---|
| Core Trail Pantry | Basic Food Stuff |
| Protein Module | Protein Kit |
| Flavor Module | Seasoning Stuff |
| Beverage Module | Drink Stuff |
| Breakfast Module | Morning Food |
| Snack Module | Trail Snack Kit |
| Weekend Backpacking Pantry | Weekend Food Box |
| Winter Pantry | Cold Weather Food Stuff |
| Emergency Pantry | Backup Food |
| Ramen Module | Noodle Kit |

## File Naming

Use lowercase kebab-case:

```text
core-trail-pantry.md
protein-module.md
flavor-module.md
weekend-backpacking-pantry.md
```

## Notes

A pantry module name should describe the purpose of the module, not just list the ingredients.
