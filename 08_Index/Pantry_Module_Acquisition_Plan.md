# Pantry Module Acquisition Plan

## Purpose

This plan defines which pantry modules should be created before building the actual `05_Pantry_Modules` layer.

Pantry modules should connect:

- Ingredients
- Recipes
- Techniques
- Trip types
- Future shopping lists
- Future meal plans

## Module Types

Use both functional modules and trip modules.

### Functional Modules

- Core Trail Pantry
- Protein Module
- Flavor Module
- Beverage Module
- Breakfast Module
- Snack Module
- Rice & Grain Module
- Ramen Module
- Soup Module
- Coffee Module
- Tea Module
- Chai Module

### Trip Modules

- Weekend Backpacking Pantry
- Car Camping Pantry
- Winter Pantry
- Emergency Pantry

## Batch 01 Recommendation

Start with the most reusable modules:

- Core Trail Pantry
- Protein Module
- Flavor Module
- Beverage Module
- Breakfast Module

## Batch 02 Recommendation

Then create trip-focused modules:

- Weekend Backpacking Pantry
- Car Camping Pantry
- Winter Pantry
- Emergency Pantry

## Batch 03 Recommendation

Then create specialized modules:

- Coffee Module
- Tea Module
- Chai Module
- Ramen Module
- Rice & Grain Module
- Soup Module
- Snack Module

## Guiding Question

For every pantry module, ask:

> If someone packed this module, what recipes and techniques would it enable?
