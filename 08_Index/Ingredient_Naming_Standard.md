# Ingredient Naming Standard v1

## Goal

Use names that are natural, searchable, and stable.

## General Rules

1. Use the common field-cooking name.
2. Prefer names a normal user would search.
3. Avoid overly technical names unless needed.
4. Use plural names for common packable ingredients when natural.
5. Keep brand names out of canonical names unless the brand is essential.

## Preferred Names

| Preferred | Avoid |
|---|---|
| Ramen Noodles | Instant Noodles |
| Chicken Packets | Pouch Chicken |
| Tuna Packets | Pouch Tuna |
| Powdered Milk | Dry Milk |
| Dehydrated Vegetables | Dried Mixed Vegetables |
| Olive Oil | Oil Packets |
| Bouillon | Bouillon Cubes |
| Tortillas | Flour Tortillas as the default name |
| Instant Rice | Minute Rice as the canonical name |

## Notes

Brand names can appear in notes or common forms, but should not usually be the canonical ingredient name.
